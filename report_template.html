Descrio do Problema

O IMC uma medida utilizada na rea de sade como um indicador simples capaz de encontrar correlaes entre o peso e altura do paciente com doenas decorrentes de obesidade, sendo inclusive capaz de classificar em diferentes grupos para obter maior preciso na anlise em questo [1], [2].

Apesar dos problemas decorrentes da simplicidade do teste [3], [4] e mtodos mais robustos tais como a bioimpedncia [5] estarem presentes, continua sendo uma medida interessante pela facilidade e baixo custo de encontr-la. A partir de tal importncia mdica, muitas questes naturalmente surgem relacionadas a testes entre diferentes populaes.

Neste estudo, sero propostos trs testes de forma a compreender possveis relaes e diferenas presentes na amostragem realizada nos anos de 2016 e 2017 no Departamento de Engenharia da UFMG. Os testes propostos so:

  • Diferena entre IMC de Homens e Mulheres;
  • Diferena de IMC entre os dois anos em estudo;
  • Diferena entre alunos de Graduao e Ps-Graduao em 2016.

Por meio destes testes, podemos obter informaes relevantes capazes de compreender diferenas baseadas em idade, sexo e ano de estudo.

Design dos Experimentos

Nos experimentos aqui realizados, procuramos extrair o mximo de informaes possveis de dados que j foram extrados. A partir disto, problemas estatsticos naturalmente iro surgir, sendo alguns deles:

  • Diferentes tamanhos de amostra;
  • Poucos dados;
  • Amostra no representativa de todo o departamento.

Apesar de tais problemas existirem, ser aproveitada ao mximo a robustez existente nos experimentos.

Diferena entre IMC de Homens e Mulheres

Dada a conhecida diferena na composio corporal entre homens e mulheres [6], torna-se relevante analisar os ndices de IMC dentro do universo em estudo. Em termos gerais, observam-se valores semelhantes de IMC entre os sexos [7], apesar das variaes na composio corporal. Para aprofundar essa anlise, podemos testar a hiptese alternativa de que o IMC mdio masculino (\(\mu_{imcm}\)) seja superior ao feminino (\(\mu_{imcf}\)). \[\begin{cases} H_0: \mu_{imcf} = \mu_{imcm}&\\H_1: \mu_{imcf}<\mu_{imcm}\end{cases}\]

Estudos referentes normalidade de ambas distribuies foram realizados, assim como um teste de comparao das varincias de Fligner-Killen.

O valor de (\(\delta^*\)) utilizado neste teste foi selecionado com base na relevncia das faixas pr-definidas de IMC, sendo que para que pessoa no meio de uma faixa precisa ter uma alterao de 2.5 em seu IMC para alternar de faixa, sendo (\(\delta^*=2.5\)).

O valor de \(\alpha=0.05\) buscou manter o padro encontrado na literatura mdica, assim como esperava-se encontrar um valor de \(\beta=0.8\) para corroborar com no encontrarmos falsos negativos.

Diferena de IMC entre os dois anos em estudo;

Realizamos para este teste dois experimentos, um para o sexo masculino e outro para o feminino. Esta estratificao baseou-se no mesmo critrio da suposta diferena dos IMCs entre os sexos.

Para cada um dos testes, iremos avaliar se (\(\mu_{2016}\)) igual a (\(\mu_{2017}\)).

\[\begin{cases} H_0: \mu_{2016} = \mu_{2017}&\\H_1: \mu_{2016} \neq \mu_{2017}\end{cases}\] Comentrios referentes ao \(\delta^*\), \(\alpha\) e \(\beta\) so anlogos aos do primeiro experimento.

Diferena entre alunos de Graduao e Ps-Graduao em 2016;

Realizamos para este experimento um teste entre a diferena da mdia do IMC entre alunos de graduao (\(\mu_{grad}\)) e ps graduao (\(\mu_{pos}\)) de forma a testar se a idade um fator que influencia a sade dos alunos do departamento. Um estudo anlogo, realizado na University of South Alabama, apresentou um IMC mdio levemente superior de alunos da ps-graduao em enfermagem, em relao aos alunos da graduao do mesmo curso [8].

\[\begin{cases} H_0: \mu_{grad} = \mu_{pos}&\\H_1: \mu_{grad} \neq \mu_{pos}\end{cases}\] Comentrios referentes ao \(\delta^*\), \(\alpha\) e \(\beta\) so anlogos aos do primeiro experimento.

Diferena entre alunos de Graduao e Ps-Graduao em 2016;

O primeiro passo carregar os dados e calcular o IMC das amostras.

imc2016 <- read.csv('imc_20162.csv')
imc2017 <- read.csv('CS01_20172.csv', sep = ';')

# Filtragem de dados
imc2016 <- imc2016[imc2016$Course == 'PPGEE',]

# Padronizao do dataframe
colnames(imc2016)[colnames(imc2016)=='Gender'] <- 'Sex'
colnames(imc2016)[colnames(imc2016)=='Height.m'] <- 'height.m'
colnames(imc2016)[colnames(imc2016)=='Weight.kg'] <- 'weight.kg'
colnames(imc2017)[colnames(imc2017)=='Weight.kg'] <- 'weight.kg'
imc2016 = imc2016[,c('Sex', 'height.m', 'weight.kg')]
imc2017 = imc2017[,c('Sex', 'height.m', 'weight.kg')]
imc2016$amostra <- '2016'
imc2017$amostra <- '2017'

# Calculando o IMC de cada populao
imc2016$imc <- imc2016$weight.kg/(imc2016$height.m^2)
imc2017$imc <- imc2017$weight.kg/(imc2017$height.m^2)

# Unio dos dataframes
imc <- rbind(imc2016, imc2017)

# Segregao de dados entre Homens e Mulheres
imc_feminino <- imc[imc$Sex == 'F',]
imc_masculino <- imc[imc$Sex == 'M',]

Em seguida, executamos o teste de Fligner-Killeen para verificar se a premissa de que as varincias so iguais se aplica no experimento.

fligner.test(imc ~ Sex, data = imc) 
## 
##  Fligner-Killeen test of homogeneity of variances
## 
## data:  imc by Sex
## Fligner-Killeen:med chi-squared = 1.0504, df = 1, p-value = 0.3054

O p-valor resultante, de 0,3054, indica que no h evidncias o suficiente para rejeitar a hiptese nula, ou seja, corrobora com a hiptese de que as varincias dos dois grupos so iguais. Executamos, ento, o teste de Shapiro-Wilker para verificar a premissa de normalidade das amostras.

shapiro.test(imc$imc[imc$Sex == "F"]) 
## 
##  Shapiro-Wilk normality test
## 
## data:  imc$imc[imc$Sex == "F"]
## W = 0.91991, p-value = 0.3179
shapiro.test(imc$imc[imc$Sex == "M"]) 
## 
##  Shapiro-Wilk normality test
## 
## data:  imc$imc[imc$Sex == "M"]
## W = 0.94947, p-value = 0.0618

Novamente, no h evidncias para rejeitar \(H_0\). Com as premissas validadas, executamos o t-test:

t.test(imc$imc ~ imc$Sex,
